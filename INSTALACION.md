# 🚀 Guía de Instalación - ShopLive

## Pasos para ejecutar la aplicación

### 1. Extrae los archivos
Descomprime la carpeta `liveshopping` en tu computadora.

### 2. Abre una terminal
- **Windows**: Presiona `Win + R`, escribe `cmd` y presiona Enter
- **Mac/Linux**: Abre la aplicación Terminal

### 3. Navega a la carpeta del proyecto
```bash
cd ruta/a/liveshopping
```

### 4. Instala las dependencias
```bash
npm install
```
⏳ Esto puede tomar unos minutos la primera vez.

### 5. Inicia la aplicación
```bash
npm run dev
```

### 6. Abre en el navegador
La terminal mostrará algo como:
```
  VITE v5.4.11  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

Abre tu navegador en: **http://localhost:5173/**

## ⚠️ Requisitos previos

Si es tu primera vez usando Node.js:

1. **Descarga Node.js** desde: https://nodejs.org/
2. Instala la versión LTS (recomendada)
3. Reinicia tu computadora
4. Verifica la instalación:
   ```bash
   node --version
   npm --version
   ```

## 🆘 Problemas comunes

### "npm no se reconoce como comando"
→ Instala Node.js desde https://nodejs.org/

### Error de permisos
→ Ejecuta la terminal como Administrador (Windows)

### Puerto 5173 ocupado
→ La app usará otro puerto automáticamente (verás el número en la terminal)

### Cámara no funciona
→ Asegúrate de dar permisos de cámara cuando el navegador los solicite
